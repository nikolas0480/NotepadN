# NotepadN: Руководство пользователя

Добро пожаловать в **NotepadN** — современный, быстрый и кроссплатформенный редактор кода, вдохновленный классическим Notepad++. Это руководство поможет вам освоиться с основными функциями программы.

---

## 1. Основной интерфейс

Интерфейс NotepadN состоит из нескольких ключевых областей:

*   **Верхняя панель инструментов (Toolbar):** Здесь расположены основные кнопки для работы с файлами (New, Open, Save), кнопка переключения режима разделенного окна (Split) и кнопка смены темы (светлая/темная).
*   **Панель вкладок (Tabs):** Отображает все открытые файлы. Позволяет переключаться между ними, закрывать и перетаскивать вкладки.
*   **Редактор (Editor):** Основная область, где вы пишете и редактируете код. Поддерживает подсветку синтаксиса, нумерацию строк и изменение масштаба.
*   **Строка состояния (Status Bar):** Находится в самом низу окна. Показывает путь к текущему файлу, его кодировку, язык программирования и количество символов.

---

## 2. Работа с файлами

### Создание нового файла
Нажмите кнопку **New** (Плюс) на верхней панели инструментов. Появится новая вкладка с названием "Untitled".

### Открытие файла
1. Нажмите кнопку **Open** (Стрелка вверх) на верхней панели.
2. В появившемся системном диалоговом окне выберите нужный файл.
3. Программа автоматически попытается определить язык программирования на основе расширения файла.

### Сохранение файла
*   Если файл уже был сохранен ранее, нажмите кнопку **Save** (Стрелка вниз).
*   Если это новый файл (Untitled) или вы хотите сохранить его под другим именем, при нажатии на **Save** откроется диалоговое окно для выбора места и имени файла.
*   *Примечание:* Звездочка (`*`) рядом с именем файла на вкладке и кнопке Save означает, что в файле есть несохраненные изменения.

### Закрытие файла
Нажмите на крестик (`X`) на вкладке файла, который хотите закрыть. Если это была последняя вкладка, автоматически создастся новый пустой файл "Untitled".

## 3. Вкладки и Разделенный вид (Split View)

NotepadN поддерживает работу с несколькими файлами одновременно и позволяет разделить окно для удобного сравнения.

### Управление вкладками
*   **Переключение:** Кликните по любой вкладке, чтобы сделать ее активной.
*   **Перемещение (Drag-and-Drop):** Вы можете перетаскивать вкладки мышью, чтобы изменить их порядок.

### Разделенный вид (Split View)
1. Нажмите кнопку **Split** на верхней панели инструментов. Окно разделится на две панели (левую и правую).
2. Вы можете перетаскивать вкладки между левой и правой панелями.
3. Каждая панель имеет свою собственную область вкладок. Активная панель подсвечивается цветной рамкой (синей в светлой теме, голубой в темной).
4. Чтобы отключить разделенный вид, снова нажмите кнопку **Split**. Все вкладки из правой панели переместятся в левую.

## 4. Настройки редактора

### Изменение языка (Подсветка синтаксиса)
Программа автоматически определяет язык по расширению файла. Если вам нужно изменить его вручную:
1. Кликните на текущий язык (например, "Language: javascript") в строке состояния (внизу справа).
2. Выберите нужный язык из выпадающего списка.
*Поддерживаемые языки:* javascript, rust, python, html, css, json, c, cpp, go, markdown, text.

### Изменение кодировки
1. Кликните на текущую кодировку (например, "Encoding: utf-8") в строке состояния.
2. Выберите нужную кодировку из списка.
*Доступные кодировки:* utf-8, windows-1251, utf-16le, iso-8859-1.
*Внимание:* При смене кодировки для уже сохраненного файла без изменений, NotepadN попытается переоткрыть файл с новой кодировкой.

### Масштаб текста (Размер шрифта)
Вы можете изменить размер шрифта в редакторе:
*   Увеличить: `Ctrl` + `Колесико мыши вверх` или `Ctrl` + `+` / `Ctrl` + `=`
*   Уменьшить: `Ctrl` + `Колесико мыши вниз` или `Ctrl` + `-`
*   Сбросить до стандартного размера (14px): `Ctrl` + `0`

### Смена темы (Светлая/Темная)
Нажмите на иконку солнца/луны в правом верхнем углу, чтобы переключиться между светлой и темной темами оформления. Выбранная тема, как и размер шрифта, сохраняются между запусками программы.

## 5. Внешние инструменты (Extensions)

NotepadN поддерживает вызов произвольных внешних программ и CLI-утилит для обработки текста прямо в редакторе (например, форматирование кода, проверка ошибок с помощью нейросетей и т.д.).

### Использование инструментов
1. Нажмите кнопку **Tools** на верхней панели инструментов.
2. В появившемся меню выберите нужный инструмент (например, "Format JSON (jq)").
3. Инструмент будет применен к **выделенному тексту**. Если текст не выделен, то инструмент обработает **весь текст** в активной вкладке.
4. Результат выполнения команды автоматически заменит исходный текст.

### Настройка своих инструментов
Список доступных команд можно настраивать через конфигурационный файл.
При первом запуске программы в директории настроек приложения (например, `~/.config/tauri-app/` на Linux, `%APPDATA%\tauri-app\` на Windows или `~/Library/Application Support/tauri-app/` на macOS) автоматически создастся файл `extensions.json`.

Вы можете отредактировать этот файл, чтобы добавить свои собственные команды:

```json
{
  "extensions": [
    {
      "id": "jq-format",
      "name": "Format JSON (jq)",
      "command": "jq ."
    },
    {
      "id": "gemini-fix",
      "name": "Gemini Fix Code",
      "command": "gemini \"исправь ошибки в коде: \""
    }
  ]
}
```

*   **id**: Уникальный идентификатор команды.
*   **name**: Название, которое будет отображаться в меню **Tools**.
*   **command**: Сама консольная команда. NotepadN передаст текст из редактора на стандартный ввод (`stdin`) этой команды и запишет её стандартный вывод (`stdout`) обратно в редактор. Вы можете использовать любые системные утилиты или цепочки команд.
